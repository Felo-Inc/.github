### Our Products

🔍 **[Felo](https://felo.ai)** — AI-powered search engine with multilingual support and real-time translation.

🌐 **[Felo Translator](https://felo.me)** — Real-time AI translation for conversations, meetings, and documents.

✨ **[Glarity](https://glarity.app)** — Browser extension for AI-powered summaries of web pages, YouTube videos, and more.

🧠 **[MemClaw](https://memclaw.me)** — Persistent project memory for AI coding agents. Creates isolated memory workspaces per project with a web dashboard for reviewing and managing what your AI remembers. Free and open source. ([GitHub](https://github.com/Felo-Inc/memclaw))
